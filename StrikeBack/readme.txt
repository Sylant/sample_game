タイトル：StrikeBack
ゲーム概要：
・プレイヤーはロボット(?)を操作して、次々と現れるエネミーとステージ上方にあるボスを倒すのは目標です。
・ボスは自身のご都度により隠すことがありますが、それでも撃殺できます。
・初期はレベルを上げることはおすすめです。

開発期間：8月中旬～
開発環境：Visual Studio 2013、Windows 10、DirectX SDK
開発人数：一人
使用言語：C++
使用ライブラリー：日本電子専門学-ゲーム制作科-学内ライブラリー
プレイ人数：一人

操作方法：

A：左へ移動
D：右へ移動
S：スキル
左クリック：確認ボタン１、一般攻撃
右クリック：攻撃２
スペース：確認ボタン２
F1：BGM　ON
F2：BGM　OFF

スキル：

・レベル３　　->　　一般攻撃パーワアップ
・レベル３　　->　　散射攻撃
・レベル５　　->　　快速移動
・レベル９　　->　　一般攻撃パーワアップ
・レベル１０　->　　散射攻撃パーワアップ
・レベル１５　->　　一般攻撃パーワアップ
             ->　　散射攻撃パーワアップ
・レベル２０　->　　散射攻撃パーワアップ

・レベル上げれば上げるほど、スキル消耗と回復は良くなります。


v1.0 2015/09/28
・攻撃したい所に左クリックまだは右クリックして、攻撃します。
・右クリックの攻撃はHPがかかります。
